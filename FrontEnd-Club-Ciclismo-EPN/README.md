# Front-End-Club-Ciclismo-EPN

Este proyecto corresponde al Frontend del sistema de gestión del Club de Ciclismo EPN.

## Getting Started with Create React App

Este proyecto fue generado utilizando [Create React App](https://github.com/facebook/create-react-app).

### Scripts disponibles

En el directorio del proyecto, puedes ejecutar:

#### `npm start`
Ejecuta la aplicación en modo de desarrollo.\
Abre [http://localhost:3000](http://localhost:3000) para verlo en tu navegador.

La página se recargará automáticamente cuando realices cambios.\
También verás errores de lint en la consola.

#### `npm test`
Lanza el ejecutador de pruebas en modo interactivo.\
Consulta [running tests](https://facebook.github.io/create-react-app/docs/running-tests) para más información.

#### `npm run build`
Construye la aplicación para producción en la carpeta `build`.\
Optimiza React en modo de producción para el mejor rendimiento.

El build está minificado y los nombres de archivo incluyen los hashes.\
¡Tu aplicación está lista para ser desplegada!

Consulta [deployment](https://facebook.github.io/create-react-app/docs/deployment) para más detalles.

#### `npm run eject`
**Nota: esta es una operación irreversible.**\
Si usas `eject`, no podrás regresar atrás.

Esto copia todas las configuraciones de Webpack, Babel, ESLint, etc., directamente a tu proyecto para tener control total.
